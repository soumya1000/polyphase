# polyphase
Phases the genotyped markers for polyploids.

Installation: This package requires the installation of boost(http://www.boost.org/),DAKOTA(https://dakota.sandia.gov/sites/default/files/docs/6.0/html-ref/index.html) and TBB (https://www.threadingbuildingblocks.org/). 

Description:
This program takes unphased haplotypes of genotyped data and the physical positions of the markers as input and returns the phased haplotypes as the output.
